## 🎨 Design System

### Colors
- **Red**: `#fef2f2` bg, `#991b1b` text - Denial input
- **Blue**: `#eff6ff` bg, `#1e40af` text - System analysis
- **Green**: `#f0fdf4` bg, `#166534` text - Generated appeal
- **Dark Blue**: `#0f172a` - Primary brand
- **Light Gray**: `#f8fafc` - Secondary sections

### Typography
- **Hero**: 48px, weight 900
- **Section Titles**: 32-40px, weight 700-800
- **Body**: 15-18px, weight 400-600
- **Font**: Inter (imported from Google Fonts)

### Spacing
- **Section Padding**: 48-72px vertical
- **Card Padding**: 24-40px
- **Max Width**: 1100px
- **Grid Gaps**: 16-32px

---

## 📱 Responsive Design

### Breakpoint: 768px

**Desktop (>768px)**:
- 2-column grids
- 4-column decision cards
- Full-width tables

**Mobile (≤768px)**:
- `.grid-2` → 1 column
- `.grid-4` → 2 columns
- `.before-after-grid` → 1 column
- `.decision-grid` → 1 column

## 📊 Content Breakdown

### Denial Examples (3 Total)

#### 1. Medical Necessity
- **Input**: "Claim denied due to lack of medical necessity. CPT code 97110..."
- **Analysis**: Denial reason, CPT flagged, documentation needed, deadline
- **Output**: Professional appeal excerpt

#### 2. Coding Error
- **Input**: "CPT code 99214 does not match diagnosis code..."
- **Analysis**: Coding mismatch, alignment issue, correction needed, filing window
- **Output**: Professional appeal addressing coding

#### 3. Prior Authorization
- **Input**: "Prior authorization not obtained before service date..."
- **Analysis**: Authorization requirement, timeline, retroactive pathway, deadline
- **Output**: Professional appeal requesting retroactive auth
