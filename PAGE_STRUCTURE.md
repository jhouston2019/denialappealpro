- **White** (`#ffffff`) - Main content sections
- **Light Gray** (`#f8fafc`) - Secondary/supporting content

## Responsive Breakpoints

### Desktop (>768px)
- 2-column grids for comparisons
- 4-column grid for decision cards
- Full-width tables

### Mobile (≤768px)
- `.grid-2` → 1 column
- `.grid-4` → 2 columns
- `.before-after-grid` → 1 column
- `.decision-grid` → 1 column
- Reduced padding on letter body

## Typography Hierarchy

### Headings
- **H1**: 48px, weight 900 (Hero)
- **H2**: 32-40px, weight 700-800 (Section titles)
- **H3**: 20-28px, weight 700 (Subsections)

### Body Text
- **Large**: 18-20px (Key messages)
- **Medium**: 15-17px (Standard content)
- **Small**: 13-14px (Supporting text)
- **Micro**: 11-12px (Labels, badges)

### Font Weights
- **900**: Hero headlines
- **700**: Section titles, emphasis
- **600**: Subheadings, strong text
- **500**: Body text, standard
- **400**: Supporting text

### Buttons
- Primary: White bg, dark text, shadow on hover
